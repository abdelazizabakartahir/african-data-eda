# African Data Exploration - EDA

## Overview
Exploratory Data Analysis on African countries using Gapminder dataset (1952-2007).

## Visualizations
- Histogram of life expectancy
- Scatter plot: GDP per capita vs Life expectancy
- Bar chart: Top 10 African countries by GDP per capita
- Box plot: Life expectancy evolution over time
- Correlation heatmap

## Tools
- Python, Pandas, Matplotlib, Seaborn
- Google Colab
